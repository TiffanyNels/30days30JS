# 30 Days of JavaScript 

A comprehensive collection of 30 mini-applications built over 30 days to master **Vanilla JavaScript**, DOM manipulation, and browser APIs. This challenge showcases a transition from foundational logic to building polished, interactive user interfaces.

## Tech Stack
- **Languages:** JavaScript (ES6+), HTML5, CSS3
- **Concepts:** DOM Manipulation, Fetch API, Local Storage, Event Handling, CSS Animations

## Project Highlights

| Day | Project | Description |
| :--- | :--- | :--- |
| 01 | **Weather App** | Real-time weather dashboard using the OpenWeatherMap API. |
| 02 | **To-Do List** | Task management with persistent data using `localStorage`. |
| 03 | **Quiz App** | Interactive quiz with dynamic state management and score tracking. |
| 04 | **Password Generator** | Secure password creator using complex string manipulation. |
| 05 | **Notes App** | A digital notepad with create/delete functionality. |
| 06 | **QR Code Generator** | Converts text or URLs into downloadable QR codes. |
| 07 | **Music Player** | Fully functional audio player with progress bars and controls. |
| 08 | **Stopwatch** | Precision timer built with `setInterval` and `clearInterval`. |
| 09 | **Calculator** | Clean, functional calculator handling complex arithmetic logic. |
| 10 | **Text-To-Speech** | Utilizing the Web Speech API to convert text input into audio. |

> *Check the folders above for all 30 projects, including: Form Validation, Drag & Drop, Dark Mode Toggle, and more.*

## Key Learning Outcomes
- **Asynchronous JS:** Managing API calls and handling JSON data.
- **State Persistence:** Keeping user data alive across page refreshes.
- **UI/UX Design:** Implementing modern, minimalist aesthetics (including Dark Mode features).
- **Tooling:** Efficient use of CSS Flexbox/Grid for responsive layouts.
